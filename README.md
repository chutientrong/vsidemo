Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Deploy on Netlify
1. Install the Netlify CLI globally

```bash
npm install -g netlify-cli
```

2. Log into your Netlify account

```bash
ntl login
```

3. Move into the site's root directory

```bash
cd vsidemo
```

4. Create a new Netlify site
